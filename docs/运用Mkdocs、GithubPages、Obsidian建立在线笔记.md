# 关于git一些基本操作

```
切换分支命令:
git checkout main
```

当你修改了仓库后，请执行以下三个命令来提交修改
注意，不用切换到`gh-pages`分支，在main分支push就好了
```
git add .
git commit -m "The information you note"
git push
```
